# 🍎 초등학생 나눗셈 연습 프로그램

과일 이모지를 사용하여 나눗셈 개념을 시각적으로 이해할 수 있도록 도와주는 교육용 프로그램입니다.

## 📸 스크린샷

```
🧺 전체 🍎 12개:
   🍎🍎🍎🍎🍎🍎🍎🍎🍎🍎🍎🍎

👉 12개를 3명에게 똑같이 나눠주면?

📦 나눠진 모습:
  [1번] 🍎🍎🍎🍎
  [2번] 🍎🍎🍎🍎
  [3번] 🍎🍎🍎🍎

💡 정답: 4개씩!
```

## ✨ 주요 기능

### 🎯 연습 모드
- **짧은 연습** (10문제) - 빠르게 연습하기
- **일반 연습** (50문제) - 충분한 연습
- **긴 연습** (100문제) - 집중 훈련

### 📊 난이도 설정
| 난이도 | 범위 | 나누는 수 |
|--------|------|-----------|
| 🌱 쉬움 | 1~10 | 5 이하 |
| 🌿 보통 | 1~20 | 5 이하 |
| 🌳 어려움 | 1~50 | 10 이하 |
| 🔥 도전 | 1~100 | 12 이하 |

### ⏱️ 타이머 설정
- ⏸️ 없음 - 시간 제한 없이 천천히
- 🐢 여유 - 문제당 30초
- 🐇 보통 - 문제당 15초
- 🚀 빠름 - 문제당 10초
- ⚡ 번개 - 문제당 5초

### 📅 일일 도전
- 매일 새로운 10문제가 출제됩니다
- 요일별로 난이도가 자동 조절됩니다
  - 월/화: 쉬움
  - 수/목: 보통
  - 금/토: 어려움
  - 일: 도전
- 연속 도전 기록이 저장됩니다

### 🏅 뱃지 시스템 (23개)
다양한 업적을 달성하고 뱃지를 수집하세요!

| 뱃지 | 이름 | 조건 |
|------|------|------|
| 👶 | 첫 발걸음 | 첫 번째 연습 완료 |
| ⭐ | 완벽한 10문제 | 10문제 100% 달성 |
| 🌟 | 완벽한 50문제 | 50문제 100% 달성 |
| 💫 | 완벽한 100문제 | 100문제 100% 달성 |
| 👑 | 연습왕 | 10회 연습 |
| 🎓 | 나눗셈 마스터 | 30회 연습 |
| 💯 | 100문제 돌파 | 누적 100문제 |
| 🔥 | 500문제 돌파 | 누적 500문제 |
| 🏆 | 1000문제 돌파 | 누적 1000문제 |
| ⚡ | 번개 손가락 | 번개모드 70%+ |
| 📅 | 첫 일일 도전 | 일일 도전 1회 |
| 🔥 | 일주일 연속 도전 | 7일 연속 도전 |
| ... | ... | 그 외 다수 |

### 📈 성장 그래프
- 정답률 변화를 시각적으로 확인
- 성장 추이 분석
- 연속 성장 기록 추적

## 🚀 실행 방법

### 요구 사항
- Python 3.6 이상
- 외부 라이브러리 없음 (표준 라이브러리만 사용)

### 실행
```bash
# CLI 버전
python3 division_practice.py

# GUI 버전 (tkinter 필요)
python3 division_gui.py
```

## 📁 파일 구조

```
division-practice/
├── division_practice.py  # 메인 CLI 프로그램
├── division_gui.py       # GUI 버전
├── CLAUDE.md            # 개발 가이드
├── README.md            # 이 파일
└── .gitignore           # Git 제외 파일
```

### 데이터 파일 (자동 생성)
- `scores.json` - 연습 점수 기록 (최근 100개)
- `badges.json` - 획득한 뱃지 목록
- `daily.json` - 일일 도전 기록

## 🎮 사용법

### 기본 조작
- 정답 입력 후 Enter
- `h` - 힌트 보기
- `q` - 종료

### 메뉴
```
1. 📅 오늘의 도전
2. 🎯 연습 시작 (50문제)
3. ⚡ 짧은 연습 (10문제)
4. 💪 긴 연습 (100문제)
5. 📊 점수 기록 보기
6. 📈 성장 그래프 보기
7. 🏅 뱃지 컬렉션 보기
8. 종료
```
